# Vehicle ECS Simulator

A C++17-based simulation of an industrial vehicle electronic control system (ECS),
designed to demonstrate embedded software architecture, diagnostics, state-machine
control, and CAN-style communication.

## Current Status
Phase 0 complete:
- Project structure initialized
- CMake build system configured
- Git repository initialized
- Basic executable builds and runs

## Planned Features
- ECS state machine
- Simulated sensors
- Diagnostics and fault handling
- CAN/J1939-style messaging
- Qt dashboard
- Documentation for architecture and testing
